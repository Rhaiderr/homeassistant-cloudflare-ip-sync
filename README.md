# Cloudflare IP Sync for Home Assistant

A custom Home Assistant integration that automatically synchronizes your public IP address with Cloudflare Rule Lists.

## Features

- Automatic public IP synchronization
- Cloudflare Rule Lists support
- WAF integration
- Zero Trust integration
- Retry and error handling
- Native Home Assistant integration
- Config Flow support
- Diagnostics
- Detailed logging

## Requirements

- Home Assistant 2026.7+
- Cloudflare Account
- Cloudflare API Token
- Cloudflare Rule List
